# Iris Flower Classification

## Overview

This project is part of the CodeAlpha Data Science Internship program. The objective is to build a machine learning model that can classify Iris flowers into their respective species based on their physical measurements.

The Iris dataset contains information about three flower species:

* Iris Setosa
* Iris Versicolor
* Iris Virginica

The model uses features such as sepal length, sepal width, petal length, and petal width to predict the flower species.

## Objective

The objective of this project is to develop a machine learning model that accurately classifies Iris flowers into their respective species using their physical measurements. The project also aims to provide practical experience with supervised learning and classification techniques.

## Dataset

The dataset was obtained using Scikit-learn's built-in Iris dataset.

### Features

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

### Target

* Setosa
* Versicolor
* Virginica

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

## Project Workflow

1. Load the Iris dataset.
2. Perform exploratory data analysis (EDA).
3. Visualize feature distributions and relationships.
4. Split the dataset into training and testing sets.
5. Train machine learning classification models.
6. Evaluate model performance using accuracy and confusion matrix.
7. Compare results and select the best-performing model.

## Results

* Successfully classified Iris flower species using machine learning.
* Achieved high prediction accuracy on the test dataset.
* Evaluated model performance using classification metrics and confusion matrix.

## Key Insights

* Petal length and petal width are the most important features for species classification.
* The dataset is well-balanced across all three flower species.
* Machine learning models can accurately distinguish between Iris species using simple measurements.
* The trained model demonstrated excellent classification performance on unseen test data.

## Conclusion

This project demonstrates the effectiveness of machine learning classification techniques for predicting Iris flower species. By utilizing flower measurements as input features, the model achieved high accuracy and successfully classified flowers into their respective categories.
